## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/rahulbala1799/qr.git
cd qr
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
